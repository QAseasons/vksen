HW_1. The first part

Сокращения
fn (file name) fol(folder name)

1. Посмореть где я
	pwd
	
2. Создать папку
	mkdir fol

3. Зайти в папку
	cd fn

4. Создать 3 папки
	mkdir fol_1 fol_2 fol_3
	
5. Зайти в любоую папку
	cd fol_1
	
6. Создать 5 файлов (3 txt, 2 json)
	touch fn{1..3}.txt fn{1..2}.json
	or
	touch fn_1.txt И_ф_2.txt fn_3.txt fn_1.json fn_2.json
	
7. Создать 3 папки
	mkdir fol_1 fol_2 fol_3
	
8. Вывести список содержимого папки
	а) При нахождении в этой папке
		ls -la
	б) При нахождении на уровень выше
		ls -la fol
		
9. Открыть любой txt файл
	nano fn.txt
	
10-11. Написать туда что-нибудь, любой текст.
	nano fn(указать расширение при наличии)
		внести изменения, нажать cntrl+X, далее Y для сохранения внесенных изменений
	
12. Выйти из папки на уровень выше
	cd ..
	
13. Переместить любые 2 файла, которые вы создали, в любую другую папку.
	a)Находясь на уровень выше
		mv fol/fn fol/fn_1 fol_2
	б)Находясь на уровень выше
		mv fol/{fn,fn_1} fol_2

14. Скопировать любые 2 файла, которые вы создали, в любую другую папку
	cp fol/{И_ф,И_ф} fol_2
	
15. Найти файл по имени
	а) Находясь в любом каталоге
	find ~/ -name fn
	б) Поиск в любом каталоге уровнем выше
	find -name fn
	
16. Просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
		tail -f fn 
		
17. Вывести несколько первых строк из текстового файла
		head -3 fn
    
18. Вывести несколько последних строк из текстового файла
		tail -3 fn
    
19. Просмотреть содержимое длинного файла (команда less) изучите как она работает.
	less fn
	:q Для выхода
	h Доступные команды
	v Для редактирования текста
	v+shift Перемещение по строке
	
20. Вывести дату и время
	Date
